本地仓库 --  远程仓库
本地仓库：
创建文件夹  --> 进入文件夹

①初始化本地虚拟仓库并提交到远程仓库：
git init       # -->生成本地   .git  文件夹  --> 本地仓库
echo "132321">>2.txt    # 输出一个文件
git add 2.txt	   # 添加文件
git commit -m "第一次提交"  #提交注释及文件到缓存区
git remote add origin    xxx@github.comxxxx  #添加远程地址
git push -u origin master  #推送到仓库


②以后所有的推送
git add 3.txt    / git add .
git commit -m "第二次提交"
git push
